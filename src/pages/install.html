Regex Search 1.8

How to use:
  • Type in a regex query and click Next to search. Clicking Next again will go to the next match. Modifying the search query will clear the current search. Note that this extension will not work on the Chrome store, so do not test it on any Chrome store pages.
Notes:
  • You can open the extension by typing Alt+Shift+F. You can go to the next and previous search results by hitting Enter and Shift+Enter, respectively. Esc clears results.
  • You can set the default case sensitivity on the options page
  • This extension uses the JavaScript regex engine. I create a regular expression using 'new RegExp(regexp, flags)', where flags are currently either 'g' or 'gi'.
  • This extension is open source and the code can be found here. I'd appreciate any help, whether it's adding new features, fixing bugs, or making the extension look better (including this page).
  • If you find any bugs, let me know here.
  • If you like this extension, please review it in the Chrome store
